^.^  claudegochi

A status line for Claude Code: how much context is left, and an ASCII cat that gets hungry as the window fills.

install github
PS C:\Users\io>

satiety = free context · hearts drain as it fills · /compact feeds the cat

Install

macOS · Linux curl -fsSL https://raw.githubusercontent.com/denipesto/claudegochi/main/scripts/install.sh | sh
Windows irm https://raw.githubusercontent.com/denipesto/claudegochi/main/scripts/install.ps1 | iex

Needs Node.js, git and the Claude Code CLI. The script clones the repo into ~/.claudegochi and writes the status line into ~/.claude/settings.json, backing it up first; nothing else is touched. To uninstall, run the same command with uninstall in place of install.

What it does

Shows the context window as a bar with the token count remaining, using Claude Code's own accounting (older versions fall back to reading the transcript). The color shifts as the window fills, so you notice before it's time to /compact.

In tamagotchi mode the numbers become a pet: it eats context, levels up over sessions, and reacts to commits and compacts. Configure widgets, language and the character with the ccg command.

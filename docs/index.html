cc-statusline

A useful status line for Claude Code. See how much context you have left — and a cat that gets hungry as the context fills up.

Install // one line

Requires Node.js, git and the Claude Code CLI. The script clones the repo into ~/.cc-statusline, writes the status line into ~/.claude/settings.json (with a backup) and leaves the rest of your config untouched. Then restart Claude Code.

What it does today

Context remaining LIVE

Parses the transcript JSONL, counts real tokens (excluding sub-agents), draws a bar and how much is left until the limit. Color shifts as it fills — so you see when it's time to /compact.

claudegochi — tamagotchi LIVE

A 3-line ASCII cat whose mood is your session state. Satiety = free context, hearts drain, he sleeps at night. Toggle it with a single field in config.json.
